# Cards
A card can be used to contain information such as text, buttons, links.


```html
<div class="card">
  
  <div class="card--header">
    <div class="row">
      <div class="column column--full">
        <h3>Default buttons</h3>
      </div>
    </div>
  </div>

  <div class="card--content"></div>
  <div class="card--footer"></div>
</div>
```
